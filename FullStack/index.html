"Full" in Full Stack mean?

            Full means that you develop both the front end, and the back end.
            The front end is what the user sees on what is called the client computer.
            The back end is what is on the server computer that delivers content to the front end.

What does "Stack" in Full Stack mean?

            Stack means that several programs, a "stack" of programs is used.
            On the front end I use HTML5, CSS3, and JavaScript.
            On the back end I use PHP, and MySQL.
            And I also have a LAMP stack on my server computer.

What does "LAMP Stack" mean?

            LAMP is a stack used for our web page server - remember, a server sends web pages to a client.
            LAMP stands for:
            L = Linux, which is the basis of our operating system
            A = Apache, which is our web server
            M = MySQL, the database we use (click here for a visual)
            P = PHP, which is the programming language we use on the server side

2. Explain MySQL: Number 2 among the most-used DB's in the entire world! Oracle is number 1 because it offers professional support, but it is a paid service. MySQL is FREE! the inventor of MySQL named this after his first daughter, her name is "My." This same inventor later created MariaDB, after his second daughter "Maria."

3. Explain SQL: Structered Query Language. MySQL is the name of the database, while SQL is the languyage you use inside of that database to answer questions -- like "which products are slow sellers..." SQL is an international standard language, works with MySQL, Oracle, SQL, Microsoft SQL Server, and many others.

4. PHP is a programming language that connects front end to back end. PHP reads HTML forms, then sends that form info to the server. PHP also reads the database and sends that back to another HTML form. PHP is a server-side language, totally secure and private, you cant see PHP when you do da "view source."

5. MySQL table: Has rows and columns, where data is organized in cells (rows are horizontal, columns are vertical). Looks like an Excel spreadsheet, but as you will learn it is actually very different.
